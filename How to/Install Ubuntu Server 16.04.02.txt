Login

Set static IP address (will take effect after reboot)
sudo nano /etc/network/interfaces
  auto eth0
  iface eth0 inet static
  address 192.168.1.100
  netmask 255.255.255.0
  gateway 192.168.1.1
  dns-nameservers 8.8.8.8 8.8.4.4

Enable firewall and allow ssh connection
sudo ufw allow ssh
sudo ufw enable

Install required packages for snapshot support if this is a Hyper-V virtual machine
sudo apt-get update && sudo apt-get -y upgrade
sudo apt-get -y install --install-recommends --allow-unauthenticated linux-virtual-lts-xenial
sudo apt-get -y install --install-recommends --allow-unauthenticated linux-tools-virtual-lts-xenial linux-cloud-tools-virtual-lts-xenial
sudo reboot
